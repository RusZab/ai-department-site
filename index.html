How it works
Как это работает
From description to production in four steps
От описания до продакшена за четыре шага
No infrastructure to manage. No models to fine-tune. Describe what you need — the agents handle the rest.
Не нужно настраивать инфраструктуру. Не нужно дообучать модели. Опишите задачу — агенты сделают всё остальное.
1

Deploy on your server

One command: docker compose up. Works anywhere Docker runs. Your infrastructure, your data, your keys.

2

Describe your project

Send a POST request in plain language. "Build a REST API for inventory management with PostgreSQL and JWT auth." That's the entire specification.

3

Agents collaborate

CTO decomposes the task. Backend Lead designs the API. Developers write code. QA creates tests. Security Lead audits vulnerabilities. DevOps handles deployment. Autonomously.

4

Get validated code

Every output goes through compilation, linting, tests, and build. Failures trigger automatic LLM-powered fixes. You get a Git branch with conventional commits, ready to merge.

Questions
Вопросы
Frequently asked
Частые вопросы
What do I need to run it?
A machine with Docker and at least one LLM API key (Anthropic, OpenAI, DeepSeek, Groq, Gemini, or local Ollama). Minimum 4 GB RAM, 2 cores. Linux, macOS, Windows with WSL.
Do you see my code or data?
No. The system runs entirely on your server. License validation is local — no phone-home, no telemetry, no data collection.
Which LLM providers work?
Six out of the box: Anthropic Claude (best quality), OpenAI / OpenRouter, DeepSeek (cheapest), Groq (fastest), Google Gemini, and Ollama (free, local). Use several at once with intelligent routing.
What if my license expires?
The system shows a warning but keeps running. Soft gate — we trust our users. Renew to get updates and support.
How much does LLM usage cost?
Depends on workload. Built-in cost controls: per-session budget (default $2), daily cap (default $50), smart routing. A typical full project build: $1–5 in API calls.
Can I customize the agents?
Yes. Agent configs are Markdown files with YAML front matter in configs/agents/. Edit, add, remove. Enterprise includes configuration support.
How do I get the key after payment?
We send the license key to your email within 1 business hour (usually minutes). Paste it into .env, restart, done.
Что нужно для запуска?
Машина с Docker и хотя бы один API-ключ от LLM-провайдера (Anthropic, OpenAI, DeepSeek, Groq, Gemini или локальный Ollama). Минимум 4 ГБ RAM, 2 ядра. Linux, macOS, Windows с WSL.
Вы видите мой код или данные?
Нет. Система работает целиком на вашем сервере. Лицензия проверяется локально — никаких обращений наружу, никакой телеметрии, никакого сбора данных.
Какие LLM-провайдеры поддерживаются?
Шесть из коробки: Anthropic Claude (лучшее качество), OpenAI / OpenRouter, DeepSeek (самый дешёвый), Groq (самый быстрый), Google Gemini и Ollama (бесплатно, локально). Можно использовать несколько одновременно с интеллектуальной маршрутизацией.
Что будет, если лицензия истечёт?
Система покажет предупреждение, но продолжит работать. Мягкая блокировка — мы доверяем пользователям. Продление даёт доступ к обновлениям и поддержке.
Сколько стоит использование LLM?
Зависит от нагрузки. Встроенный контроль расходов: бюджет на сессию ($2 по умолчанию), дневной лимит ($50), умная маршрутизация. Типичная полная сборка проекта: $1–5 за API-вызовы.
Можно ли настроить агентов?
Да. Конфигурации агентов — Markdown-файлы с YAML front matter в configs/agents/. Редактируйте, добавляйте, удаляйте. Enterprise-план включает помощь с настройкой.
Как получить ключ после оплаты?
Отправляем лицензионный ключ на вашу почту в течение 1 рабочего часа (обычно за минуты). Вставьте в .env, перезапустите — готово.
